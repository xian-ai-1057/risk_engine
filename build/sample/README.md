# Sample 同層資源

把這些檔複製到 `risk_analysis.exe` 同層（即 `dist/`）即可執行：

```
indicators_config.json     ← data/indicators_config_v3.json
risk_user_prompt.txt       ← data/prompt 的風險 user prompt
narrative_user_prompt.txt  ← data/prompt 的敘事 user prompt
tag_table.csv              ← data/tag_table.csv（選用）
```

**4 個 HTML 財報檔不在此目錄**：因財報資料敏感，請自備測試樣本。
財務概況、財務比率、現金流量、淨值調節各一份。
